# Raj Soni - Portfolio 🚀

Welcome to the source code of my personal portfolio website. I'm a Full-Stack Developer building animated web experiences, products, and AI tools.

## Instructions 🛠️

To run this project locally:
```bash
npm install
npm run dev
```

**Tech Stack** - React, TypeScript, GSAP, ThreeJS, WebGL, Tailwind, Node.js, Express, PHP, MySQL, Laravel
